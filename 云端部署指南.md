## 📋 步骤二：推送代码（1分钟）

### 2.1 打开命令行
在项目目录打开 CMD 或 PowerShell

### 2.2 配置远程仓库
**替换 `你的用户名` 为你的实际用户名**：

```bash
cd c:\Users\Administrator\新建文件夹\888
git remote add origin https://github.com/你的用户名/ethusdt-quantitative-system.git
```

### 2.3 推送代码
```bash

git push -u origin master
```

### ✅ 成功标志
看到类似输出：

## 📋 步骤三：部署到 Streamlit Cloud（2分钟）

### 3.1 访问 Streamlit Cloud

### 3.3 创建应用
点击 "New app" 按钮

### 3.4 配置应用
```

Branch: master
Main file path: app_coinex.py
```

### 3.5 部署
点击 "Deploy!" 按钮

### ⏳ 等待部署
等待 2-3 分钟，看到应用界面即成功

## 🐛 常见问题

### Q1: 推送失败
**解决方案**：
```bash
# 配置 Git 用户信息
git config --global user.name "你的用户名"
git config --global user.email "你的邮箱"

# 使用 Personal Access Token
# GitHub → Settings → Developer settings → Personal access tokens → Generate new token
```

### Q2: 仓库必须公开吗？
**是的**，Streamlit Community Cloud 免费版只支持公开仓库。Private 仓库需要 Streamlit Teams（$250/月）。

### Q3: AI 功能不工作？
**解决方案**：使用在线 API
1. 注册 DeepSeek API：https://platform.deepseek.com/
2. 获取 API Key
3. 配置到 Secrets 中
